# MemeStore

### Project Status
-- Work in Progress

A project for social media for sharing Memes templates
contains frontend and backend

## Features

- User Authentication
- Meme Templates
- User Profiles
- User Uploads
- User Followers
- User Mentions
- Post Likes
- Reply/Comments System (Nested like Twitter)
- Post abuse reporting
- Collections (User can create, edit, and delete collections of memes)
- Search Functionality
- Notifications
